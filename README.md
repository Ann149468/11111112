# README: Викторина «Memo Card»

Программный продукт представляет собой интерактивную викторину с графическим интерфейсом на базе PyQt5. Пользователь отвечает на вопросы с выбором одного правильного варианта из четырёх предложенных.

## Функциональность

Программа реализует следующие возможности:
* отображение случайного вопроса из списка;
* представление четырёх вариантов ответа (один правильный, три неправильных);
* случайное расположение правильного ответа среди вариантов;
* проверка правильности выбранного ответа;
* отображение результата (правильно/неправильно) и правильного ответа;
* подсчёт статистики: количество вопросов, правильных ответов, процент правильных ответов;
* переход к следующему случайному вопросу.

## Интерфейс

Графический интерфейс включает:
* **текст вопроса** — отображается в верхней части окна;
* **группу радиокнопок** (4 варианта ответа) — расположены в два столбца;
* **кнопку «Ответить»** — запускает проверку выбранного ответа;
* **панель результата** — появляется после проверки, содержит:
    * сообщение о правильности ответа («Правильно!»/«Неверно!»);
    * текст правильного ответа;
* **кнопку «Следующий вопрос»** — появляется после проверки ответа, запускает новый раунд.

## Технические требования

Для запуска программы необходимы:
* Python 3.x;
* библиотека PyQt5 (установка через `pip install PyQt5`).

## Установка и запуск

1. Установите зависимость:
   ```bash
   pip install PyQt5
   ```
2. Сохраните код в файл с расширением `.py` (например, `memo_card.py`).
3. Запустите программу:
   ```bash
   python memo_card.py
   ```

## Структура кода

**Основные классы:**
* `Question` — хранит данные одного вопроса: текст вопроса, правильный ответ и три неправильных варианта.

**Ключевые функции:**
* `ask(q: Question)` — отображает вопрос и варианты ответов (перемешивает варианты);
* `show_question()` — показывает панель с вопросами и вариантами ответов;
* `show_result()` — показывает панель с результатом и правильным ответом;
* `check_answer()` — проверяет выбранный ответ и обновляет статистику;
* `next_question()` — выбирает случайный вопрос из списка и отображает его;
* `click_OK()` — определяет действие по нажатию кнопки («Ответить» или «Следующий вопрос»).

**Виджеты интерфейса:**
* `QWidget` — главное окно приложения;
* `QLabel` — отображение текста (вопрос, результат, правильный ответ);
* `QRadioButton` — варианты ответов;
* `QPushButton` — кнопка управления («Ответить»/«Следующий вопрос»);
* `QGroupBox` — группировка элементов (панель вопросов, панель результата);
* `QHBoxLayout`, `QVBoxLayout` — компоновка элементов интерфейса;
* `QButtonGroup` — группировка радиокнопок для выбора одного варианта.

## Использование

1. Запустите программу. Появится окно с первым случайным вопросом и четырьмя вариантами ответа.
2. Выберите один вариант ответа с помощью радиокнопки.
3. Нажмите кнопку **«Ответить»**.
4. Программа покажет:
   * сообщение «Правильно!» или «Неверно!»;
   * правильный ответ.
5. Нажмите кнопку **«Следующий вопрос»**, чтобы перейти к новому вопросу.
6. Повторяйте шаги 2–5.

## Статистика

В процессе игры ведётся подсчёт:
* общего количества заданных вопросов (`window.total`);
* количества правильных ответов (`window.score`);
* процента правильных ответов (рассчитывается как `(window.score / window.total) * 100`).

## Добавление новых вопросов

Чтобы расширить базу вопросов, добавьте новые экземпляры класса `Question` в список `questions_list`:

```python
questions_list.append(
    Question('Текст вопроса', 'Правильный ответ', 'Неправильный 1', 'Неправильный 2', 'Неправильный 3')
)
```
